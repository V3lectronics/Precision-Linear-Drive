# Precision-Linear-Drive
Highly precise linear drive with a planetary gearbox coupled to a simple gear and clutch mechanism. The rotational motion is transferred to a screw mechanism which outputs linear motion. The motion range is limited by limit switches. The optical encoder allows for precise position feedback.
